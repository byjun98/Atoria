# Culture

Starter Android project with:

- Jetpack Compose
- Material 3
- Navigation Compose
- ViewModel + StateFlow
- Hilt
- Retrofit
- Room
- Coil

## Versions

- Android Gradle Plugin: 9.0.1
- Gradle: 9.1.0
- Compose BOM: 2026.03.00
- Activity Compose: 1.13.0
- Lifecycle: 2.10.0
- Navigation Compose: 2.9.7
- Hilt: 2.59.2
- AndroidX Hilt Compose ViewModel: 1.3.0
- Room: 2.8.4
- Retrofit BOM: 3.0.0
- OkHttp logging-interceptor: 5.1.0
- Coil: 3.4.0

## Run

1. Open this folder in Android Studio.
2. Let Gradle sync finish.
3. Select the `app` configuration.
4. Run on a device or emulator.

## Notes

- The sample network layer uses `https://jsonplaceholder.typicode.com/`.
- Room schemas are exported to `app/schemas`.
