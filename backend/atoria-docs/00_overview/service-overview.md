# 📖 서비스 개요

**아토리아**는 문화재 관광에 AI 기반 스토리텔링을 결합한
**참여형 체험 서비스**입니다.

사용자는 단순한 관람객이 아닌
**이야기의 주인공**이 되어 여행을 경험하게 됩니다.

### 🔹 서비스 흐름

1. 사용자 성향 및 스토리 컨셉 선택
2. 문화재 현장에서 미션 수행
3. 선택지 기반 스토리 진행
4. 행동 데이터 축적 (사진, 선택, 이동 등)
5. AI가 개인 맞춤형 스토리 생성
6. E-book 또는 영상 형태 결과물 제공

### 🔹 핵심 가치

* **몰입감**: 스토리 기반 체험
* **상호작용**: 미션 & 선택 구조
* **개인화**: AI 맞춤 콘텐츠 생성

👉 결과적으로
단순 관광 → **기록되고 남는 경험형 콘텐츠**로 확장

---

# ❗ 문제 정의 (Pain Points)

## 1. 수동적인 관람 경험

* 기존 문화재 관광은 정보 전달 중심
* 사용자는 단순히 ‘보는’ 경험에 머무름
* 특히 아동·가족 단위에서 몰입도 낮음

## 2. 낮은 기억 지속성

* 시간이 지나면 경험이 쉽게 잊힘
* 개인 경험이 기록으로 남지 않음
* 재회상 및 공유 어려움

## 3. 개인화 부족

* 동일한 설명과 동선 제공
* 사용자 성향 반영 부족

---

# 💡 해결 전략

## 1. 참여형 미션 기반 탐방

* 스토리 기반 미션 제공
* 사진 촬영 / 역할 수행 등 행동 중심 경험
* 능동적 참여 유도

## 2. AI 기반 결과물 생성

* 탐방 데이터 기반 스토리 생성
* E-book / 숏폼 영상 자동 제작
* 기록 + 회상 + 공유 가능

## 3. 사용자 맞춤형 경험

* 성향 기반 스토리 추천
* 개인별 다른 스토리 전개
* 동일 장소 → 다른 경험 제공

## 6. AI 스토리 생성

* 개인 맞춤형 서사 생성
* 역사 맥락 반영

## 7. 결과물 생성 및 저장

* E-book / 영상 생성
* 저장 및 공유 기능

# 📊 시장 전망

문화 관광은
**관람형 → 몰입형·개인화 체험형**으로 빠르게 변화 중입니다.

특히
**디지털 헤리티지 + AI 콘텐츠 시장**은 지속 성장하고 있으며,
체험형 관광과 콘텐츠 산업이 결합된 형태가 새로운 트렌드로 자리잡고 있습니다.

👉 아토리아는
이 흐름 속에서 **초기 시장 선점 + 확장성 확보 가능**
